# 圃場管理ダッシュボード

かっこいい圃場管理ダッシュボードアプリケーションです。温室ハウスごとの栽培状況や環境データを可視化します。

## 機能

- 温室ハウスの環境データ（温度、湿度、土壌水分量）のリアルタイム表示
- 作物情報の表示（播種日、定植日、追肥情報、収穫量など）
- 各温室ハウスの詳細情報と履歴
- 環境データのグラフ表示
- 作業タイムラインの管理
- アラート機能

## セットアップ方法

### 前提条件

- Node.js と npm がインストールされていること
- Firebase アカウントを持っていること

### インストール

1. リポジトリをクローンまたはダウンロードします

2. 依存関係をインストールします
```bash
cd farm-dashboard
npm install
```

3. Firebase設定を行います
   - `src/firebase.js` ファイルを開いて、あなたのFirebase設定情報に置き換えます
   - `.firebaserc` ファイルを開いて、あなたのFirebaseプロジェクトIDを設定します

4. ローカルで開発サーバーを起動します
```bash
npm start
```

### デプロイ方法

1. Firebaseにログインします
```bash
npx firebase login
```

2. アプリケーションをビルドしてデプロイします
```bash
npm run deploy
```

## 技術スタック

- React
- React Router
- Firebase (Firestore, Hosting)
- Tailwind CSS
- Recharts (グラフ表示)
- Lucide React (アイコン)

## ラズパイとの連携

ラズベリーパイからセンサーデータを取得するには:

1. ラズパイにセンサー（温度、湿度、土壌水分センサーなど）を接続します
2. センサーデータを読み取るスクリプトを作成し、定期的にFirebaseにデータを送信します
3. Webアプリケーションで自動的にデータが更新されます

## カスタマイズ

- `src/components/` ディレクトリ内のコンポーネントを編集することで、UIをカスタマイズできます
- `tailwind.config.js` でカラーテーマなどを変更できます
